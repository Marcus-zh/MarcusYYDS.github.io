使用方法

环境变量配置

ACCESSTOKEN

见@eallion

USE_REDIS,USE_I2C

是否使用redis和提取主题色功能
填true，则后几项必填

redis数据可先不填，后面有更快的方法

IMG2COLOR

见@anzhiyu-c/img2color-go
填入类似于https://xxx.top/api?img=的链接

KV_URL,KV_REST_API_URL,KV_REST_API_TOKEN,KV_REST_API_READ_ONLY_TOKEN

然后随便填入名字

找到.env,直接复制粘贴进去

(真的可以直接粘贴,不需要一个一个输入)

如果部署时没有填链接

域名配置

如果需要国内访问,需添加自己的域名

访问https://xxx.top/?type=complete&category=movie&page=2&theme=true得到返回即部署成功

传入

同官方

参数 是否必要 类型 默认值 可选
type 是 str 无 wishlist / progress / complete
category 否 str 全部 book/movie/tv/music/game/podcast/performance
page 否 int 1 1-最大页数
